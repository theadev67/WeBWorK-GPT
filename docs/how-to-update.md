## Updating WeBWorK-GPT

---

### If you installed via ZIP download (Option A)

There's no auto-sync, so you'll need to replace the files manually:

1. Download the latest ZIP from GitHub (same as install step 1).
2. Extract the ZIP, then **replace all files** in your original installation folder with the new ones.
3. Go to `chrome://extensions/`.
4. Click the reload button (⟳) next to WeBWorK-GPT.
5. Refresh your WeBWorK tab if one is currently open.

---

### If you installed by cloning the repo (Option B)

1. In your terminal, run:
   ```bash
   cd ~/Documents/WeBWorK-GPT  # or wherever you cloned it
   git pull
   ```
2. Go to `chrome://extensions/`.
3. Click the reload button (⟳) next to WeBWorK-GPT.
4. Refresh your WeBWorK tab if one is currently open.
